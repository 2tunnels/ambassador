Ambassador

API Gateway for Microservices

Get started


  • Designed for microservices Built on the Lyft Envoy L7 proxy, Ambassador gives you advanced load balancing features (e.g., global rate limiting), first class HTTP/2 support, and robust TLS support.
  • Integrated monitoring Automatically collect statistics and send the data to your metrics system of choice.
  • Authentication Authenticate all requests before they are proxied to your back-end services using aone of Ambassador's authentication plug-ins.
Get started in 60 seconds:
  1. After installing on Kubernetes, add a new API mapping using the REST API.
  2. Request your new service through Ambassador.
{% highlight shell %} curl -XPOST -H "Content-Type: application/json" \ -d '{ "prefix": "/v1/user/", "service": "usersvc", "rewrite": "/v2/" }' \ $AMBASSADORURL/ambassador/mapping/user {% endhighlight %}
