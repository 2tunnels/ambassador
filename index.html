Ambassador

API Gateway for Microservices

Get started


  • Designed for microservices Built on the Lyft Envoy L7 proxy, Ambassador is an elegant self-service API gateway for microservices, including robust TLS support.
  • Integrated monitoring Automatically collect statistics and send the data to your metrics system of choice.
  • Authentication Authenticate all requests before they are proxied to your back-end services using aone of Ambassador's authentication plug-ins.
Get started in 60 seconds:
1. Add your API to Ambassador.
Install Ambassador on Kubernetes, and add a new API mapping using the REST API. In this example, Ambassador will route external requests to $AMBASSADOR/v1/user/ to usersvc/v2/. 
curl -XPOST -H "Content-Type: application/json" \
      -d '{ "prefix": "/user/", "service": "usersvc" }' \
      http://localhost:8888/ambassador/mapping/user
2. Get your service URL. 
eval $(sh scripts/geturl)
3. Access your service. 
curl $AMBASSADORURL/user/health
